# jQuery

## Animações

### `.toggle()`

- Exibe ou oculta um elemento

### `.slideToggle()`

- Exibe ou oculta um elemento com um movimento deslizante

### `.fadeToggle()`

- Exibe ou oculta um elemento animando sua opacidade

### `.stop()`

- Interrompe a animação em execução antes de começar a próxima


## HTTP

### `.get()`

- `jQuery.get( url [, data ] [, success ] [, dataType ])`

### `.fail()`

- Adicione manipuladores de eventos a serem chamados quando há algum erro na requisição

### `.always()`

- Adicione manipuladores de eventos a serem chamados independentemente do resultado de uma requisição

## Geral

### `.text()`

- Tanto altera como pega o valor do texto de elementos
- Pega o conteúdo de texto de tags HTML que tem texto dentro, como as `<h1>`, `<span>` e `<p>`

### `.val()`

- Tanto altera como pega o valor do texto de elementos
- Funciona em elementos de `<input>` que são campos aonde o usuário do site insere dados, como os campos de `<input>`(todos os tipos), `<textarea>` e `<select>`

### `.on()`

- Possibilita anexar uma função de manipulador de eventos para um ou mais eventos aos elementos selecionados

### `.one()`

- Possibilita anexar uma função de manipulador de eventos para apenas 1 evento

### `.focus()`

- Usado para detectar quando o usuário entra em um campo, não necessariamente digitando

### `.attr()`

- Acessa os atributos de um elemento

### `.addClass()`

- Adiciona uma classe a um elemento

### `.removeClass()`

- Remove uma classe de um elemento

### `.toggleClass()`

- Adiciona ou remove uma ou mais classes de cada elemento no conjunto de elementos correspondentes, dependendo da presença da classe ou do valor do argumento de estado
- Se determinado elemento estiver com a classe, ela é removida. Caso não esteja com ela, ela é adicionada

### `.startsWith()`

- Comparar se a `string` de determinado elemento começa com a `string` de outro

### `.find()`

- Busca na DOM (a partir de um elemento especificado) por outro elemento

### `.mouseenter()`

- Vincula um manipulador de eventos que é disparado quando o mouse entra em um elemento ou aciona esse manipulador em um elemento

### `.mouseleave()`

- Vincula um manipulador de eventos que é disparado quando o mouse deixa um elemento ou aciona esse manipulador em um elemento
