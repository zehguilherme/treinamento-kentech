# API

## Especificação OpenAPI

Define uma descrição de interface agnóstica de linguagem de programação padrão para APIs HTTP, que permite que humanos e computadores descubram e entendam os recursos de um serviço sem exigir acesso ao código-fonte, documentação adicional ou inspeção do tráfego de rede.

Utiliza-se um arquivo `JSON` que possui todas as informações necessárias da API.

- Quem gera esse aquivo é o conjunto de ferramentas chamado [Swagger](https://swagger.io/)
  - Ajudam no ciclo de vida da API (Projeto, construção, testes, padronização, documentação)

- [Github - OpenAPI-Specification](https://github.com/OAI/OpenAPI-Specification)
